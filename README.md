# **🥜호두마켓 프로젝트**

호두마켓은 판매자와 소비자를 구별하여</br>
판매자는 상품을 등록 및 판매하고, 구매자는 상품을 구매하는 서비스입니다.

판매자로 로그인 시 상품 정보를 등록 및 수정할 수 있으며,</br>
상품의 구매는 불가능 합니다.

소비자는 상품 상세창과 장바구니를 통해 호두마켓의 상품을 구입할 수 있습니다.

</br>

# **💡 프로젝트 설명**

- 개인 프로젝트 입니다.
- 호두마켓 프로젝트는 **React**를 사용하여 API를 이용한 CRUD 구현 프로젝트입니다.</br>
